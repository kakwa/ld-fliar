Detailed Description

Represents a single file mapping entry.

This structure defines a mapping rule, including the name, matching pattern, operation type, target resource, and additional options.

Note
The array of ldfl_mapping_t structures should be terminated with an entry where name is NULL and operation is LDFL_OP_END. This sentinel entry is used to mark the end of the array.

/home/runner/work/ld-fliar/ld-fliar/lib/fliar.c.
